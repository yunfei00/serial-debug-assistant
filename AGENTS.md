# AGENTS.md

## 项目目标
这是一个基于 Python + PySide6 + pyserial 的桌面串口调试助手。

## 技术要求
- Python 3.11
- GUI 使用 PySide6
- 串口通信使用 pyserial
- 代码要清晰、可维护，适合后续扩展
- 所有注释和界面文字优先使用中文
- 不要引入不必要的第三方库

## 代码结构要求
- UI 与串口逻辑分离
- 串口操作放在 app/services/serial_service.py
- 主窗口放在 app/ui/main_window.py
- main.py 只负责启动应用

## 第一阶段功能
1. 显示串口列表
2. 选择波特率
3. 打开/关闭串口
4. 发送文本
5. 接收数据显示到文本框
6. 支持清空接收区
7. 处理基本异常，不能崩溃

## 编码要求
- 代码必须可直接运行
- 不要只写伪代码
- 改动后同步更新 README.md
- 优先做最小可运行版本，再逐步增强
